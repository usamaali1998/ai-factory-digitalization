# AI-Driven Factory Digitalization — Aira Heat Pump Manufacturing
> Case Study | Industry 4.0 | Computer Vision | Predictive Maintenance

Master's Subject project at Deggendorf Institute of Technology (2025)

## Overview
Designed an AI digitalization roadmap for a heat pump factory 
producing 250,000 units/year — targeting real-time quality control 
and predictive maintenance using IIoT + ML.

## Key Results
- €515,000 projected annual savings
- 15% scrap reduction via YOLOv8 computer vision
- 25% downtime reduction via Isolation Forest anomaly detection
- ROI within 18 months

## Tech Stack
Python · YOLOv8 · LSTM · Isolation Forest · Random Forest · 
AWS S3 · Apache Kafka · OPC UA · TensorFlow · OpenCV

## Simulations
- `simulations/anomaly_detection.py` — vibration anomaly detection
- `simulations/computer_vision_sim.py` — paint shop defect detection
